# 🚗 Big Data for Autonomous Vehicles

## 📌 Project Overview

The **Big Data for Autonomous Vehicles** project focuses on traffic sign recognition using deep learning techniques. It utilizes the German Traffic Sign Recognition Benchmark (GTSRB) dataset to train a Convolutional Neural Network (CNN) model capable of accurately classifying traffic signs. This project demonstrates how big data and artificial intelligence can support autonomous driving systems by enabling vehicles to recognize and interpret road signs in real time.

---

## 🎯 Objectives

- Develop an intelligent traffic sign recognition system.
- Train a CNN model using the GTSRB dataset.
- Improve classification accuracy through image preprocessing.
- Demonstrate the application of AI in autonomous vehicles.
- Evaluate model performance using standard metrics.

---

## 📂 Dataset

**Dataset Name:** German Traffic Sign Recognition Benchmark (GTSRB)

The dataset contains thousands of labeled traffic sign images belonging to multiple categories. It includes training and testing images captured under different lighting conditions, viewing angles, and backgrounds.
